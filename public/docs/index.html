Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Public endpoints

GET Travels

Returns paginated list of travels.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/travels?page=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/travels"
);

const params = {
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):

GET Travel Tours

Returns the paginated list of tours by travel slug.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/travels/first-travel/tours" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"priceFrom\": \"\\\"144.99\\\"\",
    \"priceTo\": \"\\\"200.30\\\"\",
    \"dateFrom\": \"\\\"2023-12-15\\\"\",
    \"dateTo\": \"\\\"2023-12-31\\\"\",
    \"sortBy\": \"\\\"price\\\"\",
    \"sortOrder\": \"\\\"asc\\\" or \\\"desc\\\"\"
}"

const url = new URL(
    "http://localhost/api/v1/travels/first-travel/tours"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "priceFrom": "\"144.99\"",
    "priceTo": "\"200.30\"",
    "dateFrom": "\"2023-12-15\"",
    "dateTo": "\"2023-12-31\"",
    "sortBy": "\"price\"",
    "sortOrder": "\"asc\" or \"desc\""
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
